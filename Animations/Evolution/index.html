EVOLUTION

Welcome! In the Animation screen, you can monitor the behaviour of the species that have evolved as they navigate the terrain. Each of these creatures have a unique brain. The brain of the best creature (according to some arbitrary metric) is displayed in the Brain dynamics screen. In the Controlpanel screen you can adjust parameters for the terrain, creature mutation and creature health. If you press space a DNA strand will appear in the control panel. Any valid DNA strand can be copied into the input box above the DNA printout box. Choosing the makeboid option in the top select then allows you (when in the animation screen) to place this creature in the terrain.

Here is a list of key controls

  • +/-
    Zoom in and out in the animation screen
  • 0/1/2/3/4
    Jump between different panes
  • Arrows
    Translate terrain in animation mode
  • Space
    Print DNA of best boid
